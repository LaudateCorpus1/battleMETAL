"map scripts"
these were added for repo tracking but are not compiled by the compiler.
Rather, these files go to specific locations in the deployment.

.ent      -> <game>/battleMETAL|iD1/maps
.rtlights -> <game>/battleMETAL|iD1/maps
.dat      -> <game>/battleMETAL|iD1/data/
.sav      -> <game>/battleMETAL|iD1/data/
